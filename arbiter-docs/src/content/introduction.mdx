# Introduction

Arbiter is a distributed API traffic control system designed to prevent abuse.

<h2 id="why-arbiter">Why Arbiter</h2>

Modern APIs face challenges like:

- traffic spikes
- abusive clients
- lack of visibility

<h2 id="features">Features</h2>

Arbiter provides:

- rate limiting
- analytics
- policy enforcement

<h2 id="architecture">Architecture</h2>

Arbiter sits between your services and incoming traffic.
